# Changelog

## Phase 1 — 2026-07-22

- สร้าง Employee Management Hub บน Firebase Project เดิม
- ใช้ Firebase Authentication และโปรไฟล์ Admin เดิม
- เพิ่ม Employee Master ที่เข้ากันได้กับ Performance Summary
- เพิ่ม Service Incentive บน Firestore
- เพิ่ม Dashboard, Export CSV และ Audit Log
- เพิ่ม Migration Center แบบเลือกไฟล์จากเครื่อง เพื่อไม่เผยแพร่ข้อมูลพนักงานบน Hosting
- สร้าง Migration Seed จากข้อมูลที่ตรวจสอบแล้ว 12 พนักงาน / 72 รายการ
- เพิ่ม Firestore Rules ที่รองรับทั้ง Web เดิมและ Web รวม
- เพิ่ม Compatibility Patch สำหรับ Performance Summary
- เตรียมหน้าและ Collections สำหรับ Workday Insight และ Monthly Performance
